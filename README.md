# Docker-installation-on-Alpine

## Primero deberemos tener instalado tanto VirtualBox com la versión que necesitemos de Alpine


![Alpine1](https://user-images.githubusercontent.com/91874398/166163907-d74c064c-897e-461c-8877-863583c846b5.png)

![Alpine2](https://user-images.githubusercontent.com/91874398/166163911-138f308b-54a8-4148-8f99-7f44889ec877.png)

## Segundo abriremos la aplicación de VirtualBox y le añadiremos un nuevo nombre a la nueva máquina virtual llamada 'alpine1'

![Alpine4](https://user-images.githubusercontent.com/91874398/166164168-fd82cec1-0ad8-4212-b035-d588678688e2.png)

## Seguidamente añadiremos la versión de Alpine a la máquina virtua, yendo a la configuración de la misma máquina y accediendo al apartado de almacenamiento, dadnole clic al disco y seleccionando el archivo correspondiente de alpine.

![Alpine5](https://user-images.githubusercontent.com/91874398/166164300-19dc78db-d022-4d11-a84f-09dfcc310ec1.png)

## Arrancaremos la máquina virtual e iremos siguiendo las diferentes indicaciones de video para asi ir configurando la maquina virtual.

![Alpine6](https://user-images.githubusercontent.com/91874398/166164358-62f4a285-3191-4450-961b-3e803ca70502.png)

![Alpine7](https://user-images.githubusercontent.com/91874398/166164360-51584de3-159d-4260-be6a-ab02f49f15d2.png)

## Acto seguido, cuando hayamos hecho las diferentes configuraciones que nos indica el video saldremos de la maquina virtual y accederemos a las configuraciones de la maquina virtual, una vez allí accedemos al apartado de red y diremos conectado a: 'Adaptador puente'.


![Alpine8](https://user-images.githubusercontent.com/91874398/166164627-50d3b00b-553b-4c91-8c92-bd5b9f8cb512.png)

## Justo después de acceder al apartado de red, accedermos al aparado de sistema donde desactivaremos las opciones de Disquete y Óptica.

![Alpine9](https://user-images.githubusercontent.com/91874398/166164677-839dea5f-a919-44e8-843e-90f7827da174.png)

## Y finalmente, volveremos a arrancar la maquina virtual para acceder a la dirección IP con la que deberemos conectar a docker.


![Alpine10](https://user-images.githubusercontent.com/91874398/166164759-12e9ce2c-68c8-46b6-a89b-35e50a53e769.png)
